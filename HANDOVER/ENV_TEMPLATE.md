# Variables de Entorno

Copia este contenido en un archivo llamado `.env` en la raíz del proyecto.

```env
# URL de tu proyecto Supabase
VITE_SUPABASE_URL=https://tu-proyecto.supabase.co

# Clave anónima (public) de Supabase
VITE_SUPABASE_ANON_KEY=tu-clave-anonima-larga

# PIN para autorizar eliminaciones críticas (ej. borrar ventas)
# Este PIN se solicita en la interfaz de usuario para acciones destructivas.
VITE_ADMIN_DELETE_PIN=1234
```
